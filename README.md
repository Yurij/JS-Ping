# JS-Ping
JavaScript Ping Emulator
![](https://find-ip.info/images/screen.png)

Demo: https://find-ip.info/en/services/ip-service/ping-traceroute

Setup & Options:

PingTest.init({
	shell: '$',
	file: 'ping.php',
	input: '#ip',
	area: '#area',
	buttstart: '#pingstart',
	buttstop: '#pingstop',
	count: 14
});
